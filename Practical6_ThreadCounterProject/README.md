# Практична робота №6

## Завдання
Розробити консольний додаток, який постійно виконує роботу в основному потоці та одночасно відслідковує натискання клавіш у фоновому потоці.

## Реалізовано

- Основний потік кожну секунду виводить лічильник: `Counter: 1`, `Counter: 2`, `Counter: 3`.
- Фоновий потік обробляє клавіші через `Thread`.
- Обробка клавіш не блокує основний потік.
- Потокобезпечність забезпечена через `lock`.

## Бінди

- `P` — пауза / продовження лічильника.
- `R` — скинути лічильник до 0.
- `C` — змінити колір тексту.
- `Q` — завершити програму.

## Як запустити

Відкрити термінал у папці проєкту та виконати:

```bash
cd ThreadCounterApp
dotnet run
```

Або відкрити `ThreadCounterProject.sln` у Visual Studio та запустити проєкт `ThreadCounterApp`.
